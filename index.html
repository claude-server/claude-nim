How It Works

Install & Launch

Run the command above. It installs the proxy, asks for your API key (first time), and launches Claude Code with the selected model.

Proxy Translates

The proxy runs on localhost and translates Anthropic Messages API calls into OpenAI-compatible requests for NVIDIA NIM.

Use Claude Code

Use Claude Code as normal. Switch models anytime with /model <name>. The proxy handles everything.

Why Claude-NIM Proxy?

Model-Family Adapters

12 built-in adapters fix per-model quirks (DeepSeek temperature caps, Llama tool_choice, Qwen stop tokens) so you don't have to.

Dynamic Model Switching

Use /model inside Claude Code to switch models on-the-fly. No restart needed.

Web Dashboard

Real-time metrics, request history, token usage, live logs, and model switching — all in your browser.

Zero Dependencies

Single TypeScript file, no Python, no Docker. Just npx claude-nim and you're running.

Security

Prompt injection scrubbing, context pruning, request body limits, encrypted key storage. No other proxy does this.

32 Tests + Stress Test

Full test suite including 100-concurrent-stream stress test. TypeScript with full type safety.
